SSAcle – SSAFY 생을 위한 스터디 관리 통합 앱

개요

SSAcle(싸클) 은 SSAFY 교육생뿐만 아니라 수료생까지, 싸피 출신이라면 누구든 참여할 수 있는 스터디 관리 통합 앱입니다.

개발 기간

2025‑01‑13 ~ 2025‑02‑21 (총 6주)

주요 서비스

  1. 스터디 개설
  2. 스터디 및 스터디원 매칭 서비스 – 관심 주제 및 모임 요일을 기반으로 추천
  3. 라이브 스터디 – WebRTC 기반 실시간 스트리밍
  4. 스터디 채팅방 – 실시간 채팅 지원
  5. AI 자기소개서 첨삭 – OpenAI API 활용
  6. AI 면접 피드백 (시선·표정 분석)

핵심 기능 상세

1. 온보딩 & 메인 화면

로그인카카오 로그인학적 인증닉네임 중복정보 입력
스켈레톤 UI최초 메인메인 화면
  • 카카오 로그인 API + JWT 인증
  • 가상의 학적 DB로 교육생/수료생 본인 인증

2. 매칭 서비스

희망 스터디 선택구인 결과스터디 찾기

필터링 + 코사인 유사도로 최적 스터디/스터디원 매칭

3. 라이브 스터디

Navbar 라이브스터디 상세라이브 스터디
  • WebRTC 실시간 스트리밍
  • MediaSoup + Socket.io 시그널링·미디어 서버

4. 스터디 단체 채팅방

Navbar 채팅스터디 채팅방

Socket.io + MongoDB 실시간 채팅

5. AI 자기소개서 첨삭

  • OpenAI API 프롬프트 엔지니어링
  • 토큰 효율화 4,500 → 1,500자
  • 수정 부분 하이라이트 + PDF 저장

6. AI 면접 피드백

  • YOLOv8n 기반 표정·동공 감지
  • 응시 비율 계산·피드백 PNG 제공
  • 온디바이스 최적화: 9초 영상 분석 100초 → 7초
